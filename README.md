# Selenium-Test-Automation
1️⃣ Contexto
Automação desenvolvida em Python para simular o preenchimento e envio
de formulários web, reduzindo tarefas manuais repetitivas.

2️⃣ Problema
Processos manuais de cadastro são suscetíveis a erro e consomem tempo
operacional.

3️⃣ Solução
Uso de Selenium para automatizar interação com página web:
- acesso à URL
- preenchimento de campos
- envio de formulário

4️⃣ Tecnologias

Python

Selenium

WebDriver
